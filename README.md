# 🌱 AgriStat Trainer (農業データ解析シミュレーター)

農業現場で取得されるデータ（発芽率、腐敗率、害虫捕獲数、発病指数など）に対する「適切な統計モデルの選択」と「結果の正しい解釈」を学ぶための新人研修用Streamlitアプリです。

単に数値を入力してp値を出すためのツールではなく、**「なぜその解析手法を選ぶのか」「なぜ他の手法ではダメなのか」**を体験を通して理解させることに特化しています。

## ✨ 主な機能

1. **データ構造に応じたモデル選択トレーニング**
   - 以下の4つの代表的な手法から、データに合ったものを選択させます。
     * 正規分布 (OLS: 通常の線形回帰)
     * 二項分布 (GLM: 発芽率など、上限のある割合データ)
     * ポアソン分布 (GLM: 捕獲数など、上限のないカウントデータ)
     * ノンパラメトリック検定 (Kruskal-Wallis検定: 発病指数などの順序データ)

2. **査読者目線の入力チェック（関所機能）**
   - 初学者が陥りやすい以下のようなミスを検知し、計算を実行する前にストップをかけて理由を解説します。
     * 原因（説明変数）と結果（目的変数）の取り違え
     * 二項分布の母数（分母）の指定忘れ
     * カウントデータ（ポアソン分布）に対する連続値やマイナス値の指定
     * 標準的なノンパラメトリック検定における複数変数の指定（二元配置の禁止）

3. **解析結果の読み方とフィードバック**
   - 不適切なモデル（割合データに対する正規分布など）を選んだ場合、結果を出すだけでなく「予測値がマイナスになる」「等分散性が崩れる」といった論文査読で指摘される致命的な問題を解説します。
   - statsmodelsのサマリー表から、現場で見るべきポイント（`coef` と `p値`）の読み方を提示します。
